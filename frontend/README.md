# BookMyEvent Frontend

Frontend for BookMyEvent, a real-time movie ticket booking platform.

## Features

* User Authentication
* Movie Booking
* Real-Time Seat Updates
* Protected Routes
* Responsive Design

## Tech Stack

* React
* Vite
* Axios
* Socket.IO

## Run Locally

```bash
npm install
npm run dev
```
